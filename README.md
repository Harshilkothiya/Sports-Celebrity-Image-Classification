# 🏅 Sports Celebrity Image Classification

> **A Computer Vision and Machine Learning application that identifies sports celebrities from images using OpenCV for face detection and feature extraction, followed by a trained classification model.**

---

## 📖 About the Project

This project classifies images of sports personalities into one of the following five categories:

🏏 Virat Kohli

⚽ Lionel Messi

🎾 Roger Federer

🎾 Serena Williams

🎾 Maria Sharapova

During the training process, **OpenCV** is used to detect and crop faces from the collected images. The processed images are then transformed into feature vectors and used to train a machine learning model capable of recognizing the respective athlete.
